# Richa Luk Thai Massage - Website Redesign

Modern, commercial-grade website for Richa Luk Thai Massage therapy business in Sandy, UK.

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Vercel Deployment

## Features
- WhatsApp integration
- AI Chatbot
- Enhanced booking system
- Modern responsive design
- SEO optimised

## Development
```bash
# Install dependencies
npm install

# Run development server (NOT USED - deploy only)
# npm run dev

# Build for production
npm run build

# Deploy to Vercel
git push origin main
```

## Deployment
Automatic deployment via Vercel on push to main branch.
